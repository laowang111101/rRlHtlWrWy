# 图书管理系统简介 python458

## 项目概述

本系统是基于Python语言和Django框架开发的图书管理系统。它提供了一个功能全面的平台，用于管理图书馆的日常运营，包括用户管理、书籍借阅、信息查询等。

## 技术栈

- **前端**：HTML、CSS、JavaScript
- **开发语言**：Python
- **后端**：Django
- **数据库**：SQLite

## 功能模块

### 登录模块

用户通过输入已注册的用户名和密码登录系统。未注册用户需先进行注册。

### 菜单功能模块

- 提供图书馆运营的全局视图。
- 查看书籍总数、会员总数、书籍类别总数、出版社总数。
- 查看最近的归还记录和重要事件。

### 借阅记录模块

允许管理员查询借阅记录的详细信息，并对其进行增删改查操作。

### 书籍管理模块

- 查看所有书籍信息。
- 对书籍信息进行增删改查操作。

### 书籍类别管理

- 查看所有书籍类别信息。
- 对书籍类别进行增删查操作。

### 出版社管理

- 查看出版社信息。
- 对出版社信息进行增删改查操作。

### 个人管理模块

允许登录用户查看和修改自己的个人信息，包括个人资料和借阅历史。

## 系统角色

系统分为两个角色：管理员和普通用户。

- **管理员**：负责系统维护、借阅记录管理、书籍信息管理等。
- **普通用户**：可以查询书籍信息、借阅书籍、查看和修改个人信息。

## 运行环境

- Python 3.*
- Django 2.*
- SQLite
